A moving front:

  set Exact solution expression  = 2-(x-t)*(x-t)
  set Right hand side expression = 2*(x-t+1)
  set Initial value expression   = 2-(x+1)*(x+1)
  set Boundary value expression  = 2-(x-t)*(x-t)

  t in [-1,1]


A steeper moving front:

  set Exact solution expression  = 2-10*(x-t)*(x-t)
  set Right hand side expression = 2*10*(x-t+1)
  set Initial value expression   = 2-10*(x+1)*(x+1)
  set Boundary value expression  = 2-10*(x-t)*(x-t)

  t in [-1,1]


Cold/hot pattern:

  set Exact solution expression  = 4*t*(y*y+y-x*x-x)
  set Right hand side expression = -4*x^2 - 4*x + 4*y^2 + 4*y
  set Initial value expression   = 0
  set Boundary value expression  = 4*t*(y*y+y-x*x-x)

  t in [0,2]
